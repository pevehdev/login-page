# TELA DE LOGIN COM VITE + REACT + FIREBASE

Página de login simples com autenticação utilzando Firebase.

<img width="900" alt="tela-login" src="https://github.com/pevehdev/login-page/blob/main/public/Login-page.PNG">

## Objetivo

Criação de tela de login com autenticação via Firebase da google.

## INSTALANDO O PROJETO EM SUA MÁQUINA

SERÁ PRECISO QUE FAÇA UM CLONE DESTE REPOSITÓRIO EM SUA MÁQUINA

```
git clone https://github.com/pevehdev/login-page.git
```

ABRA O PROJETO NO VS CODE

ACESSE A PASTA login-page PELO TERMINAL

ABRA O TERMINAL E DIGITE ESTES DOIS COMANDOS
```
npm install
```
```
npm run dev
```

IRÁ MOSTRAR NO TERMINAL O LOCAL HOST JUNTO COM A PORTA PARA ACESSAR A APLICAÇÃO LOCAL


## FERRAMENTAS QUE FORAM UTILIZADAS
- Visual Studio Code
- Git
- REACT.JS
- FIREBASE
- HTML E CSS
